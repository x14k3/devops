
## 软连接

```bash
# ############ 创建软链接
# 语法
ln -s <源文件或目录> <目标链接名>

# 为文件创建软链接
ln -s /home/user/original.txt /home/user/shortcut.txt
# 为目录创建软链接
ln -s /home/user/original_dir /home/user/shortcut_dir

# ############ 查看软链接
# 使用 `ls -l` 命令可以查看软链接，它们会显示为红色（不同终端颜色可能不同），并带有箭头指向源文件或目录
# 链接的第一个字符会显示为 `l`，而不是普通文件的 `-` 

# ############ 更新或覆盖软链接
# 如果目标链接名已存在，默认情况下会报错
# 可以使用 `-f` 参数强制覆盖已存在的链接
ln -sf /home/user/new_file /home/user/shortcut.txt

# ############ 删除软链接
# 对于文件来说，直接删除软链接并不会删除源文件；  
# 对于目录而言，直接删除软链接目录`(没有“/”)`不会删除源文件，但是若删除目录时多添加了`“/”`，那么会只保留两个空目录（源文件目录和软链接目录中文件都没了）。
# 使用 `unlink` 命令也可以删除软链接
```
