# linux 网络管理

对于Linux来说，现在流行的有网络管理有两个工具，Network服务与NetworkManager  
前者做为基础服务，桌面版和服务器中都有，后者，即NetworkManager，一般只在桌面版中安装，因为其有图形配置界面，也深受用户欢迎。  
需要注意的是，这两个网络配置，只能有一个生效，而不能同时生效。
