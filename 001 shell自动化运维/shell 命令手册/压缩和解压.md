# 压缩和解压

```bash
*.Z       使用compress压缩的文件--compress已经过时了
*.zip     使用zip压缩的文件
*.gz      使用gzip压缩
*.bz2     使用bzip2压缩
*.xz      使用xz压缩
*.tar     使用tar工具打包，没有压缩
*.tar.gz  使用tar工具打包，经过gzip压缩
*.tar.bz2 使用tar工具打包，经过bzip2压缩
*.tar.xz  使用tar工具打包，经过xz压缩
```
