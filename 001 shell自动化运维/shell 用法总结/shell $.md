# shell $?

```bash
$$   # Shell本身的PID
$!   # Shell最后运行的后台Process的PID
$?   # 最后运行的命令的结束代码（返回值） 0表示没有错误，其他任何值表明有错误)
$-   # 显示shell使用的当前选项，与set命令功能相同
$*   # 所有参数列表。如"$*"用「"」括起来的情况、以"$1 $2 … $n"的形式输出所有参数
$@   # 所有参数列表。如"$@"用「"」括起来的情况、以"$1" "$2" … "$n" 的形式输出所有参数。
$@   # 跟$*类似，但是可以当作数组用
$#   # 添加到Shell的参数个数
$0   # Shell本身的文件名
$1～$n  # 添加到Shell的各参数值。$1是第1参数、$2是第2参数…。
```
