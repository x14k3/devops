## nginx配置多个域名使用同一个端口

```nginx
    # nginx 80端口配置 （监听a二级域名）
    server {
        listen  80;
        server_name     a.com;
        location / {
            proxy_pass      http://localhost:8080; # 转发
        }
    }
    # nginx 80端口配置 （监听b二级域名）
    server {
        listen  80;
        server_name     b.com;
        location / {
            proxy_pass      http://localhost:8081; # 转发
        }
    }
```
